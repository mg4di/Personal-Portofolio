Overview

Pulse CMS is a modern headless content management system designed for development teams who need flexibility and performance. Unlike traditional CMS platforms, Pulse separates content management from presentation, allowing developers to build custom frontends while editors enjoy an intuitive content editing experience.

The platform features a component-driven editor that lets content creators build pages using pre-defined blocks, ensuring design consistency while maintaining creative freedom. GraphQL API provides efficient data fetching for any frontend framework.

The Challenge

The agency needed a CMS that could:

  • Support multiple client projects with different content models
  • Provide a visual editing experience without sacrificing developer control
  • Deliver content via API to various frontends (web, mobile, IoT)
  • Handle complex content relationships and nested structures
  • Scale efficiently as the number of clients grew

The Solution

We built Pulse CMS using Vue.js for the admin interface and GraphQL for the API layer. The component-driven editor allows content creators to assemble pages from reusable blocks, each with its own schema and validation rules.

The headless architecture means content can be consumed by any frontend — React, Vue, Next.js, or even mobile apps. Docker containerization makes deployment consistent across environments, and the multi-tenant architecture allows the agency to manage all client projects from a single instance.

Key Features

Component Editor

Visual drag-and-drop editor with reusable content blocks and live preview.

GraphQL API

Efficient, typed API with automatic schema generation from content models.

Multi-Tenant

Manage multiple client projects from a single CMS instance with isolated data.

Role-Based Access

Granular permissions for editors, authors, and administrators.

Version Control

Content versioning with rollback capabilities and change history.

Webhook Events

Trigger external workflows on content changes with configurable webhooks.
